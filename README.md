# Task

Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символа. Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма.

## solution

В качестве основного решения был использован цикл for, условием по выходу из которого была использована длина массива. Сам массив был задан изначально и прокинут в функцию.

```csharp
for (int i = 0; i < arr.Length; i++){}
```

*В качестве дополнительного решения был использован метод Where.*

## Usage

```csharp
Main(yourArray);
```

## Contributing

В качестве основного решения было решено оставить использование цикла for, т.к. с помощью него не изменялся начальный массив, и не использовались сторонние методы (кроме Append). Стоит упомянуть, что такого же функционала можно было достичь и с помощью второго решения.
